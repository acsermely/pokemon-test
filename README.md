# Pokemon game

This repo contains a webapp where users can pick their favouite pokemons.

## Run
To start both the backend and frontned run the following script:
``` sh
> npn run all
```

## Details
Some technical details about the project:
### Backend
|||
| ------ | --------|
| Server | Express |
| DB     | SQLite  |

### Frontend
|||
| --------- | ----------|
| Framework  | SvelteKit |
| Styling    | Tailwind  |
| Components | DaisyUI   |
